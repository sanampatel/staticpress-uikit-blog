# TailwindCSS Blog Template for StaticPress

This is demo blog of blog Template using TailwindCSS framework

Features of StaticPress template: 
  - Search Functionally
  - Disqus Comment System
  - Google Analytics
  - Datewise sorting
  - Git Image
  - RSS Feed
  - Sitemap
  - Category
  - Tag
